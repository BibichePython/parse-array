В этой задаче уже есть готовое решение. Но оно содержит ошибку и ваша задача найти и исправить ее. "import", "exec" и "eval" не работают в данной задаче.

Дана строка, содержащая список (list) с целыми числами или вложенными списками. Числа могут иметь знак плюс или минус перед ними. Если в строке нет списка, то вызывайте ValueError. Для строк с неверным форматом - вызывайте ValueError. Элементы списка разделяются запятыми.

Входные данные: Строка (str, unicode).

Выходные данные: Список (list) с целыми числами (int) и/или вложенными списками.

Примеры: 

parse_array("[1, 2, 3]") == [1, 2, 3]
parse_array("[[1], 2, 3]") == [[1], 2, 3]
parse_array("[-3, [-2, 0], 10]") == [-3, [-2, 0], 10]

Зачем это нужно: Эта задача для отработки ваших навыков по поиску и устранению багов. А парсер может быть улучшен и быть использован для более сложных задач в вашей практике.

Предусловия:
depth < 5
∀ x ∈ data : -1000 < x < 1000
